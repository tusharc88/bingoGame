# Bingo Game

### How to

### Important commands

###### Get source code

```
git clone https://github.com/tusharc88/bingoGame.git
cd bingoGame/client
```

###### Install dependencies

```
npm install
```

###### To start client application on port 3000

```
npm run start
```

###### To bundle application for prod

```
npm run build
```

###### To run tests

```
npm run test
```

###### To watch for tests

```
npm run test:watch
```

###### To run Storybooks

```
npm run storybook
```
Hit http://localhost:6006 to run Storybook.
